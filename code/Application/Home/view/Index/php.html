bc函数易错尝试

                    if(bcadd(0,0,2)){
                        echo "yes";
                    }else{
                        echo "no";
                    }
                    输出结果会是no；
                    因为bc函数是以字符串的形式运算的。
                    bcadd(0,0,2)，0+0='0.00'，
                    从数字角度可以等于false，但是从字符角度，它是有值的
注意：bc函数是计算后的值是string，而不是int
